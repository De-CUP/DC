Cloud & On-Premise

Cloud
IT 자원을 인터넷을 통해 필요에 따라 제공하고 사용한 만큼만 비용을 지불한다.
ex) AWS, OCI

On-Premise
자체 보유 전산실에 서버를 직접 설치하여 운영한다.

MSA 아키텍처 모델
- 작은 기능별로 서비스를 쪼개어 개발한다.
- 데이터 저장 시 하나의 DB에 중앙 집중화를 하지 않고 서비스 별 별도의 DB를 사용한다.
- API Gateway: API 서버 앞 단에서 모든 서버들의 엔드포인트를 단일화하여 묶어준다.

ELK
- Elastic 회사에서 제공하는 오픈소스이다.
- ElasticSearch: 로그 저장 및 검색
- Logstash: 로그 수집 엔진
- Kibana: 로그 시각화 및 관리

NGINX
웹 서버 구축 소프트웨어이다.

Spring Cloud Gateway
클라이언트와 백엔드 서비스 사이에 위치하는 리버스 프록시 역할을 한다.

리버스 프록시
외부 클라이언트의 요청을 받아 내부 서버로 전달하는 중계 서버이다.
보안, 로드 밸런싱, 캐싱, SSL 암호화 기능을 제공한다.
ex) Nginx, Apache, Cloudflare

ORM 프레임워크
객체와 관계형 데이터베이스의 데이터를 연결한다.

JPA
자바 ORM 기술에 대한 API 표준 명세이다.

Hibernate ORM
JPA를 자바 코드로 구현해 놓은 것이다.
