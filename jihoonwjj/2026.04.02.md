# 브라우저 주소창에 URL을 입력하면 일어나는 일

웹 브라우저가 도메인명의 IP주소를 DNS를 통해 조회한다.
DNS 서버 캐시 확인
캐시 미스시 인터넷 여러 DNS 서버를 검색될 때까지 DNS 서버에 요청

웹 브라우저가 서버와의 TCP 연결 시작
인터넷에 연결된 웹 브라우저 요청 패킷은 일반적으로 TCP/IP 전송 제어 프로토콜을 사용하여 라우터 장비, 인터넷 서비스 제공회사 교환기를 통해 이동되어, 통신 회사간 경로인 라우팅 테이블을 따라서 연결할 IP주소가 있는 웹 서버를 검색

웹 브라우저가 서버에 연결되면, HTTP, HTTPS 프로토콜에 대한 통신 규칙을 따름

웹 서버가 요청을 처리하고 응답을 다시 웹 브라우저로 전송 

웹 브라우저가 전송 받은 콘텐츠 렌더링
