# 미들웨어란

미들웨어(Middleware)는 소프트웨어 아키텍처에서 요청과 응답 사이에 끼어 들어 특정 작업을 처리하는 중간 계층이다.

## 동작 흐름

요청이 최종 목적지에 도달하기 전에 미들웨어가 순서대로 가로채서 공통 작업을 처리한다.

## 주요 역할

- **인증 / 권한 검사**: JWT 토큰 유효성 확인, 접근 권한 검증
- **로깅**: 요청 URL, 응답 상태 코드, 처리 시간 기록
- **에러 처리**: 핸들러에서 발생한 예외를 일괄적으로 처리
- **요청 파싱 / 검증**: JSON 파싱, 요청 바디 유효성 검사
- **CORS 처리**: 도메인 간 요청 허용 여부 제어
- **Rate Limiting**: 단위 시간당 요청 수 제한

 ## 핵심 정리
 
미들웨어의 핵심은 **반복되는 공통 작업을 핸들러 바깥으로 분리해 재사용 가능하게 만드는 것**이다. 덕분에 핸들러는 비즈니스 로직에만 집중할 수 있다.
