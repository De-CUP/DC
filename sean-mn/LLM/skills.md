# SKills란? - Claude Code

스킬(Skills)은 Claude가 특정 작업 성능을 향상시키기 위해 동적으로 로드라는 지침, 스크립트, 리소스가 담긴 폴더이다.

단순한 명령아가 아닌 AI 모델에게 가이드라인이나 워크플로우를 학습시켜, 특정 작업을 반복 가능한 방식으로 완료하게 만드는 '행동 지침' 역할을 한다.

<br>

## 사용 사례
- **문서 자동화** — Excel, PowerPoint, Word, PDF 등 파일 작업을 자동화한다
- **브랜드 가이드라인 적용** — 코드나 문서를 특정 스타일 규칙에 맞게 변환한다
- **팀 워크플로우 표준화** — 스프린트 계획, 코드 리뷰 등 반복 작업을 패키지화한다
- **외부 서비스 연동** — MCP 서버와 결합하여 Sentry, Asana 등과 통합한다
- **온보딩 자동화** — 신규 팀원이 프로젝트 컨텍스트를 빠르게 파악하도록 돕는다

<br>

## 스킬의 구성 요소 

| 파일/폴더 | 필수 여부 | 설명 |
|---|---|---|
| `SKILL.md` | 필수 | 핵심 지침과 워크플로우 |
| 템플릿 파일 | 선택 | Claude가 채워야 할 템플릿 |
| 예시 출력물 | 선택 | 예상 결과물 형태 |
| 스크립트 | 선택 | 실행 가능한 보조 스크립트 |
