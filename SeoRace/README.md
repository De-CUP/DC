# TIL 파일명 규칙

## 기본 형식

```
[대분류:숫자][소주제:문자2개]-YYYYMMDD.md
```

예:

```
03CD-20260403.md
```

파일명만 보고
무슨 분야인지 + 어떤 주제인지 바로 파악 가능하도록 설계했다.

## 소주제 코드 작성 규칙

소주제는 **영어 2글자 약어**를 사용한다.

가능하면:

* 단어 앞 2글자
  또는
* 핵심 자음 2개
