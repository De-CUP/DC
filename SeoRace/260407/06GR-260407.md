# い형용사 정리

## 1. い형용사란

끝이 **い** 로 끝나는 형용사.

특징:

* 명사를 바로 꾸민다 (연결어 필요 없음)
* 문장 끝에서 **だ 필요 없음**
* 단어 자체가 변형됨

예:

* 高い (비싸다)
* 新しい (새롭다)
* 難しい (어렵다)
* 面白い (재미있다)



## 2. 기본 사용

### 2.1 명사 수식

**~い + 명사**

예:

* 高い店 (비싼 가게)
* 新しい本 (새 책)
* 面白い映画 (재미있는 영화)


### 2.2 문장 끝 (현재 긍정)

형태:

~い

예:

* 値段が高い (가격이 비싸다)
* この本は面白い (이 책은 재미있다)

※ 高いだ (X)

## 3. 활용 규칙

핵심: **끝의 い 가 변한다**

### 3.1 현재 부정

い → くない

예:

* 高くない (비싸지 않다)
* 難しくない (어렵지 않다)

---

### 3.2 과거 긍정

い → かった

예:

* 高かった (비쌌다)
* 面白かった (재미있었다)

---

### 3.3 과거 부정

い → くなかった

예:

* 高くなかった (비싸지 않았다)
* 難しくなかった (어렵지 않았다)

---

### 3.4 존댓말

뒤에 **です** 추가 가능

예:

* 高いです
* 高くないです
* 高かったです

※ です 는 공손 표현, 활용 규칙 변화 아님



## 4. 예외

### いい (좋다)

원형: よい

활용:

* よくない
* よかった
* よくなかった


## 5. な형용사와 비교

| 구분    | い형용사   | な형용사   |
| ----- | ------ | ------ |
| 명사 수식 | 高い店    | 有名な店   |
| 현재 긍정 | 高い     | 有名だ    |
| 현재 부정 | 高くない   | 有名じゃない |
| 과거 긍정 | 高かった   | 有名だった  |
| 특징    | 단어가 변함 | 뒤가 변함  |


## 6. 핵심 정리

* い형용사: 단어 자체가 변화
* な형용사: 뒤 문법이 변화
