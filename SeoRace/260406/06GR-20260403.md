

# な형용사 정리

## 1. な형용사란

명사의 성질이나 상태를 설명하는 형용사.

특징:

* 명사를 꾸밀 때 **な**가 붙는다
* 문장 끝에서는 **명사처럼 활용**된다

즉,

> な형용사는 형용사처럼 쓰지만 문법적으로는 **명사에 가깝다**

---

## 2. な형용사의 기본 구조

### 2.1 명사 수식 형태

**~な + 명사**

예:

* 静か**な** 部屋 (조용한 방)
* 有名**な** 人 (유명한 사람)
* 元気**な** 学生 (활기찬 학생)

핵심:

> な = 명사를 꾸미는 연결 장치

---

### 2.2 서술형 기본 형태

현재 긍정:

~ **だ**
~ **です**

예:

* 彼は有名**だ**
* この町は静か**です**

문장 끝에서는 **명사처럼 동사(이다)가 붙는다**

---

## 3. 시제 변화 규칙

### 3.1 현재형

~が **だ**
~が **です**

의미:

현재 상태 설명

---

### 3.2 과거형

~が **だった**
~が **でした**

의미:

과거 상태 설명

예:

* 昔は有名**だった**
* 子供のころ元気**でした**

---

### 3.3 부정형

~ **じゃない**
~ **ではありません**
~ **じゃありません**

의미:

상태 부정

예:

* 有名**じゃない**
* 元気**ではありません**

---

### 3.4 과거 부정

~ **じゃなかった**
~ **ではありませんでした**

예:

* 静か**じゃなかった**
* 便利**ではありませんでした**



## 4. 예외 사항

### 4.1 おなじ (同じ)

의미:

같은

특징:

**な가 붙지 않는다**

예:

* 同じ 人
* 同じ 方法

> 일부 な형용사는 문법 규칙을 따르지 않는다


## 5. 활용 구조 요약

| 형태    | 구조                    |
| ----- | --------------------- |
| 명사 꾸밈 | ～な + 명사               |
| 현재 긍정 | ～だ / ～です              |
| 현재 부정 | ～じゃない / ～ではありません      |
| 과거 긍정 | ～だった / ～でした           |
| 과거 부정 | ～じゃなかった / ～ではありませんでした |
| 예외    | 同じ                    |


## 6. 핵심 정리

な형용사 =

형용사처럼 보이지만 **명사처럼 움직인다**

그래서

* な 붙는다
* だ 붙는다
* 부정형도 명사 패턴 따라간다
